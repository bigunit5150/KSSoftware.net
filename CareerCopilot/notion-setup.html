Set Up Your
Notion Workspace

Create the three databases that power your job search system. This takes about 10 minutes and only needs to be done once.

Before starting this step, make sure you have a Notion account and have installed the Chrome extension from Step 1 . Don't have a Notion account? Create a free Notion account →

Databases

Three databases, one setup.

Your job search system uses three Notion databases that work together. You'll create all three in the same Notion page or workspace.

Database 1 — Job Tracker
Required

Stores every job you save, tracks status, holds the JD text, and receives Claude's AI analysis.

  1. Open Notion and navigate to the page where you want your job search hub
  2. Type /database and select "Table — Full page"
  3. Name it exactly: Job Tracker
  4. Add the following properties:
Property Name Type
Role Title Title (default)
Company Text
Job URL URL
Raw JD Text
Status Select
AI Notes Text
Resume Text
Location Text
Job Type Select
Salary Range Text
Date Added Date
Date Applied Date
ATS Source Text
Notes Text

For the Status select field, add these options: Bookmarked, AI Reviewed, AI Reviewed - Skip, Applying, Applied, Interviewing, Offer, Closed

For the Job Type select field, add: Remote, Hybrid, Onsite

Database 2 — Resume Repository
Required

Stores all your resume variants and selection rules. Claude reads from this database to pick the right resume for each role.

  1. Create another full-page table database in the same workspace
  2. Name it exactly: Resume Repository
  3. Add the following properties:
Property Name Type
Resume Name Title
Category Select
Version Text
Last Updated Date
Resume Content Text
Active Select
Notes Text

Category options: Director, Senior Manager, VP, AI/Technical Consulting, Other

Active options: Yes, No

Database 3 — Interview Prep Hub
Required

Tracks every interview opportunity with research briefs, cheat sheets, mock interview notes, and post-interview reflections.

  1. Create another full-page table database in the same workspace
  2. Name it exactly: Interview Prep Hub
  3. Add the following properties:
Property Name Type
Opportunity Title
Company Text
Role Title Text
Status Select
Claude Project URL URL
Job Tracker URL URL
Resume Used Text
Interview Round Select
Interview Date Date
Cheat Sheet Text
Company Research Text
Key Talking Points Text
Questions To Ask Text
Mock Interview Notes Text
Post Interview Notes Text
AI Coaching Notes Text

Status options: Active Prep, Interview Scheduled, Interview Complete, Offer, Closed

Interview Round options: Phone Screen, Recruiter Screen, Hiring Manager, Panel, Final Round, Offer Stage

Connector Setup

Connect Claude to your workspace.

Claude needs permission to read and write your Notion databases. This is a one-time step done inside your Claude account settings. This enables the official Notion MCP connector used by Claude Projects and Claude Code.

01

Open Claude Settings

Go to claude.ai and sign in. Click your profile icon in the top-right corner. Select Settings from the dropdown menu.

02

Navigate to Integrations

In the Settings sidebar, click Integrations . You'll see a list of available connectors.

03

Connect Notion

Find Notion in the integrations list and click Connect . You'll be redirected to Notion to authorize access.

04

Authorize Your Workspace

In the Notion authorization screen, select the workspace that contains your three job search databases and click Allow access . You'll be redirected back to Claude automatically.

05

Verify the Connection

Back in Claude, the Notion integration should now show as Connected . You're ready to move to Step 3.

Configuration

Grab your database IDs for the extension.

The Chrome extension needs two values from Notion: your integration token and your Job Tracker database ID. Here's how to find both.

Integration Token

  1. Go to notion.so/profile/integrations
  2. Click "New integration" (or find your existing one)
  3. Give it a name like "Job Search AI"
  4. Copy the Internal Integration Token
    It starts with secret_
  5. Paste it into the Chrome extension settings

Database ID

  1. Open your Job Tracker database in Notion (full page view)
  2. Copy the URL from your browser address bar
  3. The database ID is the 32-character string in the URL:
    notion.so/ 320037c8790b80049498d26d7d85322c ?v=...
  4. Paste it into the Chrome extension settings
Keep your Integration Token private. It gives write access to your Notion workspace. Never share it or commit it to a public repository.
