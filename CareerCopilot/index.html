The System

What each phase does.

Phase 1

Capture

Install the Chrome Extension. When you find a role on any job board, click the extension icon to save the full posting — Role Title, Company, Job URL, Raw JD, Location, Job Type, Salary Range, and ATS Source — directly to Notion.

Phase 2

AI Job Review

Claude reads Bookmarked jobs, evaluates fit against your Resume Selection Rules, selects the right resume variant, optimizes it for the JD, and writes AI Notes, Resume, and updated Status back to Notion.

Phase 3

Interview Prep & Coaching

One chat per opportunity. Claude generates an 8-section master cheat sheet, runs mock interviews, prepares interviewer-specific briefs, and debriefs after each round — building a living document that grows with each interaction.

How to Run It

Three execution modes.

Claude Projects

Interactive mode at claude.ai. Create a project, connect Notion, and chat your way through job reviews and interview prep. Best for hands-on setup and first-time users.

Interactive Setup guide →

Claude Code

CLI and IDE automation via Notion MCP. Run job review workflows from your terminal or editor with the job-review-workflow skill. Best for developers.

CLI / IDE Setup guide →

OpenClaw

Always-on agent with cron, heartbeat, and chat triggers. Automatically reviews new bookmarked jobs on a schedule without manual intervention.
