Installation

Up and running
in four steps.

01

Install from Chrome Web Store

Click the button above or search "Job Post Tracker" in the Chrome Web Store. Click "Add to Chrome" and confirm the installation.

02

Pin the Extension

After install, click the puzzle piece icon in your Chrome toolbar. Find "Job Post Tracker" and click the pin icon so it's always visible.

03

Open Settings & Enter Your Credentials

Click the extension icon, then click "Settings" at the bottom of the popup. You'll need two values: your Notion Integration Token and your Job Tracker Database ID. If you already copied your Database ID in Step 1 , paste it here. For both values, see the Configuration section below for detailed instructions on where to find them.

04

Save Your First Job

Navigate to any job posting on LinkedIn, Greenhouse, Lever, or any job board. Click the extension icon. The job title, company, and full job description are saved to Notion automatically.

Configuration

Grab your credentials for the extension.

The Chrome extension needs two values from Notion: your integration token and your Job Tracker database ID. Here's how to find both.

Integration Token

  1. Go to notion.so/profile/integrations
  2. Under Internal Integrations , click New integration (or select an existing one)
  3. Name it "CareerPilot"
  4. Select your CareerPilot workspace from the Associated workspace dropdown, then click Create
  5. Click Configure integration settings
  6. Open the Content access tab, click Edit access , select CareerPilot from the Private dropdown, and save
  7. Open the Configuration tab and click Show next to Internal Integration Secret — the token starts with ntn_
  8. Copy the token and paste it into the Chrome extension settings

Database ID

  1. Open your Job Tracker database in Notion (full page view)
  2. Copy the URL from your browser address bar
  3. The database ID is the 32-character string in the URL:
    notion.so/ 320037c8790b80049498d26d7d85322c ?v=...
  4. Paste it into the Chrome extension settings
Keep your Integration Token private. It gives write access to your Notion workspace. Never share it or commit it to a public repository.

Generic fallback

For job boards without a dedicated parser, the extension uses a generic extractor. You can optionally add your own Anthropic API key (BYOK) in the extension settings to enable Claude-powered extraction on unknown boards. The key is never sent to any server other than Anthropic's API.
