Installation

Up and running
in four steps.

01

Install from Chrome Web Store

Click the button above or search "Job Post Tracker" in the Chrome Web Store. Click "Add to Chrome" and confirm the installation.

02

Pin the Extension

After install, click the puzzle piece icon in your Chrome toolbar. Find "Job Post Tracker" and click the pin icon so it's always visible.

03

Connect Your Notion Account

Click the extension icon for the first time. You'll be prompted to enter your Notion Integration Token and the ID of your Job Tracker database. Complete Notion setup (Step 2) first if you haven't already — you'll need those values.

04

Save Your First Job

Navigate to any job posting on LinkedIn, Greenhouse, Lever, or any job board. Click the extension icon. The job title, company, and full job description are saved to Notion automatically.

Configuration

Two fields to fill in.

Notion Integration Token

Where to find it: Notion Settings → Connections → Develop or manage integrations → New integration → copy the Internal Integration Token

secret_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Notion Database ID

Where to find it: Open your Job Tracker database in Notion → copy the URL → the database ID is the 32-character string before the ?

320037c8790b80049498d26d7d85322c

Generic fallback

For job boards without a dedicated parser, the extension uses a generic extractor. You can optionally add your own Anthropic API key (BYOK) in the extension settings to enable Claude-powered extraction on unknown boards. The key is never sent to any server other than Anthropic's API.
