Always-On Agent

Set Up
OpenClaw

Run CareerCopilot as an always-on agent. Automatically review new jobs on a schedule and interact via your favorite chat app.

OpenClaw is an open-source AI agent that runs locally on your machine. It uses Claude as its backend and loads the same AgentSkills from CareerCopilot's skills/ directory. Unlike Claude Code, OpenClaw can run on a schedule and connect to your chat platforms — processing new jobs automatically without any manual trigger.

Installation

Install OpenClaw.

01

Install the CLI

curl -fsSL https://openclaw.ai/install.sh | bash macOS users can also download the companion app from the GitHub releases page.

02

Configure Claude Backend

OpenClaw supports multiple AI backends. For CareerCopilot, configure it to use Claude. Follow the OpenClaw getting started guide to set up your API key and model preferences.

03

Connect Notion

OpenClaw connects to services through its own integration system. Configure Notion access following the OpenClaw docs so the skills can read and write to your Job Tracker, Resume Repository, and Interview Prep Hub.

04

Load Skills

Clone the CareerCopilot repo and open it in OpenClaw. The skills in skills/ are automatically discovered. You can also copy them to ~/.openclaw/skills/ for global access.

Automation

Always-on job review.

OpenClaw can run the job-review-workflow skill on a schedule. When you bookmark a job via the Chrome extension, OpenClaw picks it up on the next cycle — evaluates fit, optimizes your resume, and writes results back to Notion automatically.

Configure cron or heartbeat triggers in OpenClaw to set the review frequency. See the OpenClaw skills documentation for scheduling details.

Chat Integration

Interact from anywhere.

OpenClaw connects to your existing chat platforms. Ask it to review jobs, build interview prep, or check status from whichever app you already use.
